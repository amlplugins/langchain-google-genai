# @amlplugins/langchain-google-genai

Native `@langchain/google-genai` SDK re-exported under the `@amlplugins` namespace.

> LangChain Google Generative AI provider — ChatGoogleGenerativeAI with Gemini models and embedding integration.

## Install

```bash
npm install @amlplugins/langchain-google-genai
```

## Use

```ts
import { /* named exports from @langchain/google-genai */ } from "@amlplugins/langchain-google-genai";
// or
import sdk from "@amlplugins/langchain-google-genai";
```

This package additionally re-exports `@langchain/core` under the `secondary` namespace.

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
